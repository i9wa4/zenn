# Zenn CLI

[![Ask DeepWiki](https://deepwiki.com/badge.svg)](https://deepwiki.com/i9wa4/zenn-i9wa4)

## 1. 初回セットアップ

```console
npm install
zenn init
```

## 2. プレビュー

```console
zenn preview
```

以下にアクセスする

<http://localhost:8000>
